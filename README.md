# WSACA Tools

A collection of free, web-based tools developed by the **Washington State Association of County Assessors** to support our members and the homeowners we serve.

These tools are open source and free to use. Any Washington county assessor's office is welcome to link to them directly, embed them on their own website, or adapt them for local needs.

---

## Available Tools

### Property Tax Exemption Screening Tool

A guided screening tool that helps Washington homeowners check whether they may qualify for the expanded **Property Tax Exemption for Senior Citizens and People with Disabilities** under the program changes taking effect in tax year 2027.

The tool walks users through eligibility requirements, calculates an estimated combined disposable income using an abbreviated version of DOR's Form 63 0036, and returns a screening result tailored to their county's income thresholds.

**[→ Open the screening tool](exemption-screening-tool.html)**

Built to reflect the program changes enacted by Engrossed Substitute Senate Bill 6162 (2026), including the new income thresholds for tax years 2027–2029, the standard deduction option, the consolidated State School Levy, the rental income deduction, and the updated benefit amounts at each level.

This is a screening tool only — it does not make official determinations of eligibility. Final eligibility is always determined by the applicant's county assessor.

## About This Project

These tools were developed collaboratively by WSACA members with AI assistance, and are made available freely to the assessor community and the public. We believe modern tooling should be a shared resource across all 39 counties — not something every office has to build independently.

If you're a county assessor's office and would like to:

- Link to a tool from your county's website
- Embed a tool in your own page
- Suggest improvements or report a bug
- Contribute a tool of your own

— please reach out, or open an Issue on this repository.

---

## Maintenance

Tools in this repository are maintained on a best-effort basis by WSACA leadership. Income thresholds, statutory references, and program details will be updated as the Department of Revenue publishes revised data and as legislation changes the program.

If you spot something that's out of date or incorrect, the fastest way to flag it is to open an Issue using the **Issues** tab above.
